# GameForge Token Site

Static landing page for the GameForge Token research project.

## About

GameForge Token is a pre-mainnet research project exploring blockchain-based infrastructure for contribution tracking, attribution, and coordination in decentralized game development. The project is in active research and development. No token is currently live, and no smart contracts have been deployed to mainnet.

## Contents

This repository contains the static site files for the project landing page:

- `index.html` — Landing page
- `styles.css` — Stylesheet
- `CNAME` — Custom domain configuration

## What This Repository Is Not

- This is not a token sale or fundraising platform
- This repository does not contain smart contracts or protocol code
- This site does not collect user data or use analytics

## Deployment

The site is deployed via GitHub Pages with a custom domain (`gameforgetoken.com`).

No build step is required. Files are served as static assets.
